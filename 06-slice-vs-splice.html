splice() vs slice()

  • splice()

    The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

    Example:

    const arr = [1, 2, 3, 4, 5];
const removed = arr.splice(2, 2, 'a', 'b', 'c');
console.log(arr); // [1, 2, 'a', 'b', 'c', 5]
console.log(removed); // [3, 4]

  • slice()

    The slice() method returns a new array containing elements from the original array. You can specify the start and end indices of the slice. The original array is not modified.

    Example:

    const arr = [1, 2, 3, 4, 5];
const sliced = arr.slice(2, 4);
console.log(sliced); // [3, 4]
console.log(arr); // [1, 2, 3, 4, 5]
