Array Methods

JavaScript arrays have a number of built-in methods that make it easy to work with arrays. Here are some of the most commonly used methods:

  • concat()
  • sort()
  • reverse()
  • splice()
  • slice()

concat()

concat() is used to merge two or more arrays. It does not modify the original arrays, but returns a new array that contains the elements of the original arrays. Here's an example.

sort()

sort() is used to sort the elements of an array in place. By default, it sorts the elements in alphabetical order. However, you can pass a function as an argument to customize the sorting behavior. Here's an example.

reverse()

reverse() is used to reverse the order of the elements in an array. It modifies the original array. Here's an example.

splice()

splice() is used to add or remove elements from an array. It modifies the original array. You can use it to add new elements, remove existing elements, or both. Here's an example.

slice()

slice() is used to create a new array that contains a portion of the original array. It does not modify the original array. You can use it to extract a range of elements from the array. Here's an example.
